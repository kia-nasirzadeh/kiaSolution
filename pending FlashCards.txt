---------------Daily check:
1- ssh-keygen -t rsa -b 4096 -C "mail"
2- pub -> ~/.ssh/authorized_keys
3- nano /etc/ssh/sshd_config -> PubkeyAuthentication yes
4- service ssh restart
---------------Details:
1- build ssh keys:
ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
(t=>type, b=>size of sshKey in bits, c=>comment to add to your pubkey)
2- in your server, copy and paste public key into "~/.ssh/authorized_keys" file
3- in your server, open "/etc/ssh/sshd_config" file and uncomment "PubkeyAuthentication yes"
4- in your server, "service ssh restart"

cpanel:
how to manual deploy with cpanel?
------
------
------
1- in the root of your website make a pub_html folder & and put all of website files in it
2- make .cpanel.yml in the root of your website files:
---
deployment:
  tasks:
    - export DEPLOYPATH=/home/<your-user-name>/public_html/
    - /bin/cp -r pub_html/* $DEPLOYPATH

3- put your site on github
4- add github repo to your cpanel (git version control)
3- "pull or deploy"->"update from remote"->"deploy HEAD commit"
